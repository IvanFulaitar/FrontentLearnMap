# Джерела зображень

Цей файл документує кожне зображення, додане до курсу «Робоче середовище» (vscode-setup).

## Підсумок

- **19 зображень — власні скріншоти** (`vscode-*.png`, реальний VS Code, зроблені автором курсу на своєму Windows-комп'ютері). Це найвищий пріоритет джерела за інструкцією проєкту.
- **macOS-встановлення — без зображення.** Автор курсу не має Mac, тому реального macOS-скріншота встановлювача немає. Замість фейкового чи запозиченого скріншота текст лесона прямо каже: процес аналогічний Windows (сайт → застосунок → перетягнути в Applications), скріншотів немає.
- Зовнішніх hotlinks, знайдених пошуком чи скопійованих із чужих сайтів, — **немає**.

## Формат і розташування

Усі файли — `.png`, розташовані в `public/images/courses/vscode/`. `LessonFigure` обмежує CSS max-width реальною шириною файлу (через `width`/`height` пропси), щоб браузер не розтягував і не розмивав скріншоти на широких екранах.

## Обмеження

- **macOS**: жодного зображення немає (пояснено вище) — урок «Встановлення VS Code» тепер показує лише 4 реальні Windows-скріншоти з номерованими підписами (без окремого текстового списку кроків і без перемикача Windows/macOS, оскільки для macOS немає що перемикати).
- Кілька старих SVG-схем із попередньої версії курсу (структура папок, Open Folder, панель термінала, вікно Format On Save) видалено — їх повністю замінили реальні скріншоти або (для терміналу) двоє нових реальних скріншотів користувача.
- Курс скорочено з 10 до 8 уроків: уроки «Типові проблеми» і «Фінальна перевірка» (разом з їхніми зображеннями `vscode-common-problems.png` і `vscode-final-checklist.png`) видалено з курсу.

**Зовнішніх hotlinks у навчальному контенті курсу «Робоче середовище» не використано.**
